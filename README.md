# Web-Dev-Project
🎬 Movie Catalog

📌 Description

Movie Catalog is a web application built with Angular that allows users to browse, search, and save their favorite movies.

Users can:  • View a list of movies  • Search movies by title  • See detailed information about each movie  • Add or remove movies from favorites
